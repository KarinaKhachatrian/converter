## Инструкция по использованию программы «PDF Конвертер» для ОХЛП

Программа предоставлена в формате **.exe** в директории проекта **output**. Она предназначена для запуска на ОС семейства **Windows**.

Для работы программы необходимо установить **Pandoc**, поскольку он используется в модулях обработки файлов.

Ссылка для установки: https://github.com/jgm/pandoc/releases/latest

Для работы с программой необходимо запустить установочный файл и следовать шагам в окне обработчика:
1. Выбрать директорию с PDF-файлами ОХЛП, нажав соответствующую кнопку
2. При верном выборе директории будет отображено соответствующее сообщение. Далее нажать на кнопку «Конвертировать файлы»
3. Конвертация запустится автоматически, у пользователя будет возможность просмотреть лог выполнения. Также предоставлена возможность отмены обработки.

Файлы на выходе будут сохранены в выбранной директории с PDF-файлами в дочерней директории «HTML». Для каждого HTML-файла создается своя директория с исходным файлом и изображениями.
